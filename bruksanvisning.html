Bruksanvisninger

Git

  1. Lag repo i en mappe med "git init"
  2. Legg til filer med "git add . (eller spesifiser filnavn)"
  3. Legg til filer i databasen med "git commit -m "(melding)""
  4. Hent en ny database med "git clone (link til database)"
  5. Skyv endringer til en fjerndatabase med git push

Kate editor

Kate er en flott editor for tekst og kode, og de har mye god dokumentasjon på siden deres.
